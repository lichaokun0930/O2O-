# AI分析两个版本对比指南

## 🎯 核心区别

### 📊 版本1: 基础GLM模式
- **启动**: `启动Dashboard.bat`
- **知识注入**: 固定3000字符全量知识
- **优势**: 快速启动,简单够用
- **Token**: ~3200/次

### 🚀 版本2: 向量检索增强模式  
- **启动**: `启动Dashboard_AI增强版.bat`
- **知识注入**: 智能检索~2500字符相关知识
- **优势**: 分析精准,Token省37%
- **Token**: ~2000/次

## ⚡ 快速对比测试

```powershell
# 双击运行
AI对比测试.bat
```
