## 📝 技术细节

### CSS加载策略
```
1. 中国CDN (bootcdn.net) - 最快
2. 国际CDN (jsdelivr.net) - 备用
3. Dash Bootstrap - 第三备份
4. 内联CSS - 最终兜底
```

### 响应式实现方式
```css
/* Bootstrap Grid + Flexbox */
.row { display: flex; flex-wrap: wrap; }
.col-xl-2 { flex: 0 0 16.666667%; max-width: 16.666667%; }
```
