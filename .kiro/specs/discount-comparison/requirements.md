# 折扣商品供给与销售分析 - 对比模式

## 背景
当前"折扣商品供给与销售分析"模块已升级为 ECharts 版本，展示本店各一级分类的：
- 分类SKU总数（紫色宽柱）
- 折扣SKU数（青绿色窄柱）
- 折扣渗透率（橙色折线）

现需要增加对比模式，支持与竞对门店进行折扣策略对比分析。

## 用户故事

### US-1: 折扣渗透率差异对比
**作为** 门店运营人员
**我希望** 看到本店与竞对在各分类的折扣渗透率差异
**以便** 快速识别折扣策略差距较大的分类

**验收标准:**
- [ ] 显示各分类的折扣渗透率差异（本店 - 竞对）
- [ ] 差异 > 10% 用绿色表示（本店折扣更多）
- [ ] 差异 < -10% 用红色表示（竞对折扣更多）
- [ ] 差异在 ±10% 内用灰色表示（基本持平）

### US-2: 折扣SKU数量对比
**作为** 门店运营人员
**我希望** 对比本店与竞对在各分类的折扣SKU数量
**以便** 了解双方的折扣商品供给规模

**验收标准:**
- [ ] 分组柱状图展示本店和竞对的折扣SKU数
- [ ] 本店用青绿色，竞对用灰色
- [ ] 支持 tooltip 显示详细数据

### US-3: 综合对比卡片
**作为** 门店运营人员
**我希望** 看到折扣策略的整体对比概览
**以便** 快速了解双方折扣策略的整体差异

**验收标准:**
- [ ] 显示整体折扣渗透率对比（本店 vs 竞对）
- [ ] 显示折扣SKU总数对比
- [ ] 显示折扣渗透率最高/最低的分类

---

## 设计方案讨论

### 方案A: 折扣渗透率差异对比（差异柱状图）
**展示内容:** 各分类的折扣渗透率差异值（本店 - 竞对）
**优点:** 直观显示差距，一眼看出哪些分类需要调整
**缺点:** 只展示差异，看不到绝对值

### 方案B: 折扣SKU数量对比（分组柱状图）
**展示内容:** 本店和竞对的折扣SKU数并排展示
**优点:** 可以看到双方的绝对数量
**缺点:** 不同分类SKU基数不同，难以直接比较

### 方案C: 综合对比（推荐）
**展示内容:**
1. 顶部：对比卡片（整体折扣渗透率、折扣SKU总数）
2. 图表1：折扣渗透率对比（双折线图或分组柱状图）
3. 图表2：折扣SKU数量对比（分组柱状图）

**优点:** 信息全面，既有整体概览又有分类细节
**缺点:** 占用空间较大

### 方案D: 镜像柱状图（参考动销率对比）
**展示内容:** 本店折扣渗透率在左，竞对在右，中间是分类名
**优点:** 对比直观，与现有动销率对比风格一致
**缺点:** 只能展示一个指标
